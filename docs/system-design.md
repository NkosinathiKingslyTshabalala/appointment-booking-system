# Appointment Booking System

## Project Overview

The Appointment Booking System is a full-stack web application that allows clients to discover service providers, browse available services, and book appointments online.

The platform also enables service providers to manage their profiles, services, schedules, and appointments, while administrators oversee the overall system.

---

## Objectives

- Simplify appointment scheduling
- Prevent double bookings
- Allow providers to manage availability
- Provide a secure authentication system
- Support multiple user roles
- Deliver a responsive user experience
