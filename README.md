# Appointment Booking System

A full-stack appointment scheduling platform.

## Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Node.js
- Express
- TypeScript

### Database
- PostgreSQL
- Prisma ORM


## Features

- User authentication
- Role based access control
- Service provider management
- Appointment scheduling
- Availability management
- Notifications
